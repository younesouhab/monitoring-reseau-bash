# Monitoring réseau automatisé (Bash)

Script permettant de tester la connectivité réseau :

- DNS Google (8.8.8.8)
- Cloudflare (1.1.1.1)
- Passerelle locale (192.168.1.1)
- Domaines publics (google.com, github.com)

## Objectif
Diagnostiquer rapidement l’état du réseau en environnement Helpdesk.

## Utilisation
chmod +x monitoring.sh
./monitoring.sh
